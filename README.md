### [002 - Encyclopedia of Parentheses](https://atcoder.jp/contests/typical90/tasks/typical90_b)
- bit全探索が使える問題
  - n個の要素からなる集合の部分集合を全て調べ上げる手法のこと
- bit全探索を用いて、2^N通りのパターンを考え、その中で、条件を満たしているものだけを出力する．
